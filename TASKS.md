# Project Roadmap: Tuition Class Management System

This document outlines the remaining features and modules to be implemented to complete the Tuition Class Management System.

## 1. Class & Subject Management

- [ ] **Subject Management**: Create, edit, and delete subjects (e.g., Mathematics, Science).
- [ ] **Grade/Batch Management**: Manage different grades (e.g., Grade 10, Grade 11, A/L).
- [ ] **Class Creation**: Link Subjects + Grades + Teachers to create a "Class".
- [ ] **Timetable/Schedule**: Define days and times for each class.

## 2. Student Management

- [ ] **Student Registration**: Detailed profile (Parent contacts, School, Address).
- [ ] **Class Enrollment**: Assign students to specific classes.
- [ ] **Student ID Cards**: Generate digital or printable ID cards with QR codes.
- [ ] **Student Portal**: A separate view for students to see their classes and materials.

## 3. Attendance System

- [ ] **Mark Attendance**: Manual entry or QR code scanning for class entry.
- [ ] **Attendance History**: View attendance records by student or by class.
- [ ] **Absence Notifications**: Auto-send SMS/Email to parents if absent (optional).

## 4. Exam & Results Management

- [ ] **Create Exams**: Schedule exams for classes.
- [ ] **Enter Marks**: Teachers enter marks for students.
- [ ] **Progress Reports**: Generate PDF report cards.
- [ ] **Leaderboards**: Show top students (configurable privacy).

## 5. Learning Materials (LMS Features)

- [ ] **Upload Materials**: Teachers upload PDFs, Tutes, and Past Papers.
- [ ] **Video Links**: Share recording links for missed classes.
- [ ] **Student Access**: Students can download materials for their enrolled classes.

## 6. Fee Management (Optional/If needed)

- [ ] **Fee Structure**: Define monthly fees per class.
- [ ] **Payment Recording**: Mark students as "Paid" for the month.
- [ ] **Payment Receipts**: Generate digital receipts.
- [ ] **Due Payments**: List students with pending fees.

## 7. Staff/Teacher Management

- [ ] **Teacher Profiles**: Manage teacher details and qualifications.
- [ ] **Teacher Assigning**: Assign teachers to subjects/classes.
- [ ] **Staff Access Control**: Specific permissions for non-admin staff.

## 8. Dashboard Enhancements

- [ ] **Real-time Stats**: Active students today, income this month, etc.
- [ ] **Upcoming Schedule**: Widget showing today's classes.
- [ ] **Recent Activities**: Log of system actions (for security).

## 9. System Settings

- [ ] **Institute Details**: Name, Logo, Contact info updating.
- [ ] **Theme Customization**: Accent colors, Dark/Light mode toggle persistence.
- [ ] **Backup**: Data export functionality.
